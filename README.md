# 🎬 Movie Search Autocomplete App

A fast and interactive movie search app built using vanilla JavaScript.  
It fetches real-time movie data from the OMDb API and provides a smooth autocomplete experience with keyboard navigation.

---

## 🚀 Features

- 🔍 Real-time movie search (debounced input)
- ⚡ Fast API handling with AbortController (cancels old requests)
- ⌨️ Keyboard navigation (Arrow Up / Down / Enter)
- 🖱️ Click to select movie
- 🧠 Caching using Map (avoids duplicate API calls)
- 🎯 Smooth scrolling to active selection
- 🌙 Dark mode friendly UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OMDb API
